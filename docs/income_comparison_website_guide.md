## Quick Reference

### Data Source
Statistics Canada Census 2021 - Employment Income (2020 reference year)

### Key Percentiles (Canada-wide)
- 10th: $5,200
- 25th: $16,900
- 50th (Median): $37,358
- 75th: $67,800
- 90th: $102,000
- 95th: $129,700
- 99th: $216,200

### Total Recipients
~27 million Canadians with employment income
